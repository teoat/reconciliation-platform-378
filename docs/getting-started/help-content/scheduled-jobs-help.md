# Scheduled Jobs Help

## Overview

Scheduled jobs allow you to automate reconciliation processes by running them on a schedule. You can configure recurring jobs with custom schedules.

## Getting Started

1. **Create Scheduled Job**
   - Navigate to Reconciliation Settings
   - Click "Scheduled Jobs" tab
   - Click "New Job"
   - Configure schedule

2. **Configure Schedule**
   - Select frequency (daily, weekly, monthly)
   - Set execution time
   - Choose data sources
   - Configure notifications

3. **Monitor Jobs**
   - View job execution history
   - Check job status
   - Review execution logs
   - Handle failures

## Common Tasks

- **Job Scheduling**: Create and configure scheduled jobs
- **Schedule Management**: Edit or delete schedules
- **Execution Monitoring**: Monitor job execution
- **Failure Handling**: Configure failure notifications and retries
- **Job History**: Review job execution history

## Troubleshooting

**Issue**: Job not executing
- **Solution**: Verify schedule is active and time zone is correct

**Issue**: Job failing
- **Solution**: Check data sources, configuration, and execution logs

**Issue**: Notifications not received
- **Solution**: Verify notification settings and email configuration
