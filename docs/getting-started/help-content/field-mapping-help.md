# Field Mapping Help

## Overview

Field mapping allows you to map source data fields to standard reconciliation fields. This ensures consistent data structure across different sources.

## Getting Started

1. **Access Field Mapping**
   - Navigate to Data Ingestion page
   - Select uploaded file or data source
   - Click "Field Mapping" tab

2. **Map Fields**
   - Review source fields
   - Select target field for each source field
   - Configure field transformations
   - Set default values if needed

3. **Save Mapping**
   - Review mapping summary
   - Save mapping configuration
   - Apply to other files if needed

## Common Tasks

- **Auto-Mapping**: Use automatic field detection and mapping
- **Custom Mappings**: Create custom field mappings
- **Field Transformations**: Apply transformations (date formats, currency, etc.)
- **Mapping Templates**: Save and reuse mapping configurations
- **Validation**: Validate mapped fields before processing

## Troubleshooting

**Issue**: Fields not mapping correctly
- **Solution**: Check field names and data types match expected format

**Issue**: Transformation errors
- **Solution**: Verify transformation rules and data format compatibility

**Issue**: Mapping lost after refresh
- **Solution**: Ensure mapping is saved before navigating away
