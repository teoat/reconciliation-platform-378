# Discrepancy Resolution Help

## Overview

Discrepancy resolution helps you identify and resolve differences between matched records. You can create resolution actions and track resolution progress.

## Getting Started

1. **View Discrepancies**
   - Navigate to Reconciliation Results
   - Click "Discrepancies" tab
   - Review discrepancy list

2. **Resolve Discrepancies**
   - Select discrepancy to resolve
   - Review discrepancy details
   - Choose resolution action
   - Add resolution notes

3. **Track Resolution**
   - Monitor resolution progress
   - Review resolution history
   - Export resolution reports

## Common Tasks

- **Discrepancy Analysis**: Analyze discrepancy patterns and causes
- **Resolution Actions**: Create and apply resolution actions
- **Bulk Resolution**: Resolve multiple discrepancies at once
- **Resolution Templates**: Use templates for common resolutions
- **Resolution Tracking**: Track resolution status and history

## Troubleshooting

**Issue**: Discrepancies not appearing
- **Solution**: Ensure reconciliation has completed and matches exist

**Issue**: Cannot resolve discrepancy
- **Solution**: Check permissions and ensure discrepancy is in resolvable state

**Issue**: Resolution not saving
- **Solution**: Verify all required fields are filled and try again
