# File Upload Help

## Overview

The file upload feature allows you to upload data files for reconciliation. Supported formats include CSV, Excel, and JSON files.

## Getting Started

1. **Upload a File**
   - Navigate to the File Upload page
   - Drag and drop files or click to browse
   - Wait for upload to complete
   - Review upload summary

2. **File Requirements**
   - CSV files must have headers in the first row
   - Excel files should have data starting from row 1
   - JSON files should be valid JSON format
   - Maximum file size: 100MB per file

3. **Verify Upload**
   - Check file preview
   - Verify column names
   - Review data quality indicators
   - Confirm row counts

## Common Tasks

- **Bulk Upload**: Upload multiple files at once
- **File Validation**: Automatic validation of file format and structure
- **Data Preview**: Preview data before processing
- **Error Handling**: Review and fix upload errors
- **File Management**: View, delete, or re-upload files

## Troubleshooting

**Issue**: File upload fails
- **Solution**: Check file size, format, and network connection

**Issue**: Columns not recognized
- **Solution**: Ensure first row contains headers and format is correct

**Issue**: Data appears incorrect
- **Solution**: Check file encoding (UTF-8 recommended) and delimiter settings

**Issue**: Upload timeout
- **Solution**: Split large files or increase timeout settings
