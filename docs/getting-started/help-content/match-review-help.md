# Match Review Help

## Overview

Match review allows you to review and validate reconciliation matches. You can approve, reject, or modify matches before finalizing results.

## Getting Started

1. **Access Match Review**
   - Navigate to Reconciliation Results
   - Click "Review Matches" button
   - View match list

2. **Review Matches**
   - Review match confidence scores
   - Compare matched records
   - Verify match accuracy
   - Approve or reject matches

3. **Finalize Review**
   - Complete review of all matches
   - Save review decisions
   - Proceed to discrepancy resolution

## Common Tasks

- **Bulk Approval**: Approve multiple matches at once
- **Match Comparison**: Compare matched records side-by-side
- **Confidence Review**: Review and adjust match confidence scores
- **Match Modification**: Modify or split matches
- **Filtering**: Filter matches by confidence, status, or criteria

## Troubleshooting

**Issue**: Cannot approve matches
- **Solution**: Ensure you have review permissions

**Issue**: Match details not loading
- **Solution**: Check data source connectivity

**Issue**: Review progress lost
- **Solution**: Save review decisions regularly
