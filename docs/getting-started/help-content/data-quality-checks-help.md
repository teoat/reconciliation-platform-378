# Data Quality Checks Help

## Overview

Data quality checks help ensure your data meets quality standards before reconciliation. You can configure validation rules and review quality reports.

## Getting Started

1. **Access Quality Checks**
   - Navigate to Data Ingestion
   - Click "Quality Checks" tab
   - Review quality indicators

2. **Configure Validation Rules**
   - Define validation rules
   - Set quality thresholds
   - Configure auto-fix options
   - Save configuration

3. **Review Quality Reports**
   - View quality summary
   - Review quality issues
   - Fix data quality problems
   - Re-run quality checks

## Common Tasks

- **Validation Rules**: Configure data validation rules
- **Quality Metrics**: Monitor data quality metrics
- **Issue Resolution**: Fix data quality issues
- **Quality Reports**: Generate quality reports
- **Auto-Fix**: Configure automatic quality fixes

## Troubleshooting

**Issue**: Quality checks not running
- **Solution**: Verify validation rules are configured and enabled

**Issue**: Too many quality issues
- **Solution**: Review and adjust validation rules or fix source data

**Issue**: Quality reports not generating
- **Solution**: Check data availability and report configuration
