## Archive Structure

Duplicate and outdated files have been moved to:
- `docs/archive/project-management/complete/` - Completion reports
- `docs/archive/project-management/summary/` - Summary files
- `docs/archive/project-management/status/` - Status files
- `docs/archive/project-management/progress/` - Progress reports
- `docs/archive/project-management/agent/` - Agent-specific files
- `docs/archive/project-management/plans/` - Plans and roadmaps
- `docs/archive/project-management/reports/` - Reports and diagnostics
- `docs/archive/project-management/misc/` - Miscellaneous files

See [CONSOLIDATION_COMPLETE.md](./CONSOLIDATION_COMPLETE.md) for details.

## Documentation Principles

1. **Single Source of Truth**: One document per topic
2. **Active Status**: Only current, relevant documents in main directory
3. **Archive Old**: Move completed/outdated docs to archive
4. **Cross-Reference**: Link related documents
5. **Status Indicator**: ⭐ marks primary status document
