# Diagnostic Prompt Quick Reference

**Last Updated**: 2025-01-16

---

## Which Version to Use?

### Use V1 (Original) When:
- You need a quick diagnostic without automation
- You want manual control over all fixes
- Historical tracking isn't needed
- Team collaboration features aren't required
- Business analysis isn't needed

### Use V2 (Combined) When:
- You want automated fixes for common issues
- You need historical tracking and trends
- You want predictive insights
- Team collaboration is important
- Business ROI analysis is needed
- You want interactive remediation workflow
